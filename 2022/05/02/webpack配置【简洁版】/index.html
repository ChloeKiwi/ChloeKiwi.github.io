webpack配置【简洁版】

  • 目录 dist src js
  • 全局/局部安装好webpack
    • npm install webpack@3.6.0 –save-dev—>node_modules
    • npm install：—>nodemodules
  • webpack.config.json
    • 写好出入口
  • npm init安装各种包（path）–》package.json
  • 安装两个loader
    • npm install –save-dev css-loader
    • npm install –save-dev style-loader
    • 如果安装的webpack是低版本：
      • npm install css-loader@2.0.2 –save-dev
        npm install style-loader@0.23.1 –save-dev
  • 配置style、css—>use:[]
  • npm run build

  • 安装less

    • npm install –save-dev less-loader less
    • npm install –save-dev less-loader@4.1.0 less

  • 图片的依赖

    • npm install --save-dev url-loader

    • npm install –save-dev file-loader@3.0.1

  • npm install –save-dev babel-loader@7 babel-core babel-present-es2015
  • npm install –save-dev @babel/preset-es2015
  • npm install –save-dev @babel/preset-es2015
  • 配置vue:
    • npm install vue –save
