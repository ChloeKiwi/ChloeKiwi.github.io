内核过程

IB assume：

  • graph-structured data makes model training in the IB principle hard（Not independent/identically distributed)

GIB did:

  • extract minimal + sufficient information from the graph structure &node features
    • hierarchically extract information followed Markov chain
    • local-dependence assumption:
      • only correlate with neighbors
  • derive variational bounds for GIB
    • making GIB tractable and amenable for the design andoptimization of GNNs.
    • variational upper bound : minimizing redundant information
    • variational lower bound : sufficient information used to predict

GIB is:

  • a representation learning on graph-sturctured data

How GIB works :

  • applying it to GAT
    • leverage the attention weights of GAT to sample the graph structure
    • 2 sampling algorithms:

      • categorical distribution

      • Bernoulli distribution
  • 2 models:
    • GIB-Cat
    • GIB-Bern

GIB wants to:

  • minimize information
    • difficulties : correlation between data points
  • maximize prediction
  • robustness ↑
  • against attack ↑
  • classification accuracy ↑ under adversarial perturbation（compares to other defense models)

Graph Representation Learning is:

  • node+edge==>adjacency matrix [A]
  • node attributes==>X
  • input=(A,X)

GIB-Cat & GIB-Bern works:

  • refine the graph structure using the attention weights (GAT)
  • refines node representations ()
  • Why they defense:
    • depend loosely on th graph structure
      • [A] is only used to decide the potential neigh-bors for each node
      • message passing based on ZA()

What the author did:

  • test whether GNNs trained with the GIB objective are more robust and reliable
  • compare GIB-Cat and GIB-Bern with baselines including GCN and GAT
    • GCN ok
    • GAT no
