基于半监督学习爬虫识别方法的研究与实现

基于半监督学习爬虫识别方法的研究与实现-高航-笔记

1、背景

  • 网站承受海量级访问流量负载
  • 要识别网络爬虫+减轻网站负载压力

现有的网络爬虫技术

传统的启发式规则方法流程式地判别网络爬虫

but

传统的启 发式的网络爬虫识别技术已经难以有效地识别隐藏较深的网络爬虫

现有：

使用最广泛的是【有监督学习方式的机器学习方法】，

  • 需要大量的有标注的数据集，通过有监督的方式训练分类器从而完成分类
    • 能够得到较好的分类效果
  • 但在训练集的标注工作上将耗费大量的人力、分类器泛化能力较差

所以

够同时利用有标注的数据和无标注的数据训练分类器的半监督学习方法成为了新的研究方向。

以日志形式存储的 Web 访问请求数据以及网络爬虫行为与正常

文本处理中向量化技术

鉴别网络爬虫—》

  • 更好滴利用爬虫技术
  • 组织恶意爬虫带来的危害

爬虫识别技术：信息封闭

爬虫技术：发展开放成熟

机器学习方法：

  • 文本分类
  • 日志异常检测
  • 爬虫识别技术上准确率较好

有监督学习方法

  • 有监督训练分类器
  • 需要大量的带有标签的数据集
    • 人力投入
    • 结果模型泛化能力差

半监督学习

  • 同时利用有标签的数据和无标签的数据训练文本分类器
  • 找出数据之间的联系，同时利用上述两种特征信息
  • 提高爬虫识别方法的精确率
  • 解决训练数据样本不足
  • 人工标注数据费时费力

第二章 相关理论与关键技术概述

1、网络爬虫技术概述

  • 本质：脚本程序
  • 根据既定的规则对万维网上的资源进
    行访问
  • 对访问的资源进行过滤、下载并保存以完成数据采集
  • 分类：
    • 通用网络爬虫
      • 依次访问 URL 相对应的网页
      • 在爬取网页的同时发现新的链
        接进行解析得到新的 URL
      • 将其添加到待爬取列表
      • 去掉网页中的所有标签仅保留文本信息，将这些文本信息保存到相应的数据库存储归档
      • 重复上述步骤遍历
    • 主题网络爬虫
      • 为了抓取用户感兴趣的某一主题相关的网页而产生
      • 针对性更强
      • 有选择地访问与预设主题相关的网页
      • 如何分析并识别与预设主题相关的网页是主题网络爬虫的核心机制
    • 深层网络爬虫
      • 对象一般是动态的网页
      • 无法通过静态链接获取
      • 需要附上表单信息才能获取到
      • 与通用网络爬虫的区别：
        • 是否需要提交表单信息

2、网络爬虫核心技术

模拟浏览器

  • 编写脚本程序控制浏览器的行为以实现数据的爬取
  • 网页对应的 JS、CSS 等文件都会被执行并下载

第三章 基于半监督学习的日志分类模型

1、数据预处理

日志数据集

  • 网站访问日志
    • 记录每个IP访问的足迹
    • 数据预处理-》会话切分-》利用日志相关特征建图-》数据的词向量
  • HTTP访问记录日志
    • 以W3C日志文件格式居多
    • 由以下字段组成：客户端IP地址
      • 包含：日期+时间+时区的时间戳+……

会话切分

  • 将日志文本文件切分为一个会话列表
  • 会话：
    • 一组连续的具有相同访问行为的请求组成
    • 会话切分就是将【网络日志切分为多个会话】
    • 基于时间的启发式会话识别方法中实验效果最好的一种作为实验的日志会话切分方法

将具有相同用户代理的请求和 IP 地址前两段相同的访问记录划分为单
个用户

日志文本向量化

  • 构建词向量字典
    • ​ Word2Vec中的Skip-gram模型
  • 获取日志文本的词向量
  • 设计出一种基于滑动窗口日志建立邻接矩阵的方法建图
  • 使用图卷积神经网络来对建立好的图进行模型的半监督训练

建图

  • 文本分类的算法
  • 文本分类领域中基于神经网络结构的 Word2Vec 模型
  • 常用于情感分析等应用
  • 每条日志用向量组表示，每组向量组之间使用 TF-IDF 关键词计算权重，再将每组向量组联系起来

Word2Vec模型

  • 获取词向量的工具包
  • 将一个词转化为向量
  • 是一个简单的神经网络模型
  • 输入：高维向量—-》包含相邻词关系的输出向量
    • 权重矩阵！
    • 训练过程中的参数！
  • CBOW 模型：
    • 输入：目标词
    • 输出：上下词语的词向量
  • Skip-gram模型：
    • 输入：上下文词语
    • 输出：目标词的词向量

日志文本向量化

  • 文本是自然语言处理领域中的核心

  • 所有的NLP都围绕文本开展

  • 为什么向量化：

    • 让计算机读懂文字

  • 自然语言数据vs日志数据

    • 自然：

      • 由不同的单词组成

    • 日志：

      • 由单词、数字和特殊字符构成

      • 表示计算机服务器的访问记录或系统进程记录

      • 形式是由人在机器上预设的模板打印生成的，是给人看的

      • W3C格式的日志文件：

        • 有几列特征以数字的形式存在

          • 响应状态码
          • 服务器发送的字节数

        • 以时间序列存在的时间特征

          • 可以转换为以数字形式存在的时间戳

    • 将日志看成文本，对日志进行向量化处理

      • 日志文本向量化：
        1. 构建词向量字典
          1. 本文中采用 Word2Vec 中的 Skip-gram 模型来构建词向量字典
          2. Skip-gram模型的目的在于：学习更新权重矩阵
        2. 根据已构建的词向量字典获取日志文本的词向量

  • 输出层和输入层都是一维向量
  • 最后通过Softmax Regression 计算概率
    • 得到输入节点的相邻节点的概率分布
  • 目的：
    • 学习更新权重矩阵
    • 第 i 个单词的权重由 W 权重矩阵的第 i 行表示
  • 我们所需要的词向量就是该模型隐藏层的权重矩阵
  • 最后可以得到：
    • 整个词向量字典在每条日志中形成的词向量
    • 还需要对该词向量进行 TF-IDF的处理！（👇关键词权重计算算法）

one-hot向量

  • 稀疏矩阵，维度灾难
  • 语义缺失
    • 这种向量表示失去了词语之间的相似性

数字归一化

  • 目标：减小单个特征维度的跨度
  • 只留下有用的信息
  • 如果要对日志进行建图，以向量的形式表示每条日志和每条日志之间的关系，需要对这些数字形式存在的数据进行数据标准化的处理
  • 让不同纬度之间的特征在数值上具有一定的比较性
  • 最常用的方法：
    • min-max标准化【本文采用】
      • 离差标准化
      • 对原始数据的线性变换，使得结果落在[0,1]区间内
      • min 为样本数据的最小值，max 为样本数据的最大值

对日志进行建图

  • 以向量形式表示每条日志和日志之间的关系
    • 需要对数据（数字形式）进行数据标准化处理（👆数字归一化）

关键词边权重算法TF-IDF

  • 常用于资讯检索和**文本挖掘**的加权技术

  • TF ：词频

    • 若某一词语出现的频率较高，即 TF（Term Frequency）较高

  • IDF：逆向文件频率

    • 在其他的文本中出现较少，即 IDF（Inverse Document Frequency）较高

  • 文本—》日志

    • TF-IDF算法可以运用到日志处理中

    • 日志处理：

      • 将整个日志文件看成所有文本文件的总
        和
      • 单条日志看成单个文件
      • 日志中的每个特征则是文本中的关键词

    • 文本之间的相似度计算：

      • 使用余弦相似度来表示每条日志向量两者之间的边权重
      • 余弦相似度的值越大就表示两者越相似
      • 余弦值越接近于 1，则表示两个向量的夹角越小，两个向量的方向越接近，即越相似；
      • 似度越高，即两个向量顶点之间的边权重值越大

    • 向量总数：访问日志的总条数

    • 余弦值的范围为[-1,1]，我们需要将该值归一化到[0,1]

    • 相似度的范围就转变为[0,1]

    • 结合日志的相关特征和 TF-IDF 算法

    • 计算出每条日志向量组之间的边权重，从而计算出每条日志之间的关联

日志建图算法

  • 得到以每条日志为单位的向量
  • 每条日志之间的边权重

图卷积神经网络GCN

  • Graph：图论中的用顶点和边建立相应关系的拓扑图
  • 与传统CNN的区别：
    • 传统的 CNN 虽然可以处理图片等欧式结构的数据
    • 处理社交网络等非欧式结构数据却较难
      • 图片：一般结构完整，由工整的矩阵组成
      • 社交网络：图论中的图，由顶点和边建立起的拓扑图
        • 反映了节点之间的相似性
        • 每个单数据样本就是一个节点，通过节点之间的相似性训练出更好的有助于提升分类效果的模型

图卷积神经网络算法

  • 现如今有两大主流方法
    • 基于空间的方法
      • 核心：将图数据规则化
      • 利用节点直接相连的邻居节点提取特征
    • 基于频谱的方法
      • 核心：利用谱图理论直接处理图数据
      • 通过傅里叶变换或者拉普拉斯变换，借助图的拉普拉斯矩阵的特征值和特征向量研究图的性质
      • 得到用以识别分类网络爬虫的模型

半监督图卷积神经网络模型

  • 核心：利用基于图的半监督学习方法进行节点分类的工作
  • 实质：将图卷积的核心理解为特征传播
  • 从
  • 基于频谱图卷积改进提出的方法
  • 频谱方法：
    • 直接对图结构数据和节点特征进行卷积操作

在每个卷积层只处理一阶邻居特征，利用分层传播叠加多个卷积层来建立基于图卷积的神经网络模型以达到多阶邻居特征传播
