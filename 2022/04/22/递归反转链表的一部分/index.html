注意点

1、递归函数要有 base case

if (head.next == null) return head;

只有一个节点的时候反转也是它自己，直接返回即可

二、反转链表前 N 个节点

将链表的前 n 个节点反转

明确递归函数的定义**：
  • **输入一个节点 head**和需要转换的个数
  • 将「以 head 为起点」的链表反转，到第n-1个
  • 从head.next开始递归
  • 最后要处理头节点的连接
    • 2指向1
    • 1指向successor
  • 返回反转之后的头结点 ，整个反转的最后一个last

注意点

1、base case 变为 n == 1

  • 只反转前一个元素，就是头元素
  • 记录后驱节点

2、head 节点在递归反转之后不一定是最后一个节点

  • 要记录后驱 successor（第 n + 1 个节点）
  • 反转之后将 head 连接successor(而不是null)
